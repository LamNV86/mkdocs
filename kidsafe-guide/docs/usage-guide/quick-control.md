## Bật/tắt các tính năng

Sau khi liên kết thành công điện thoại với máy tính, ở tab “Điều khiển” có thể bật/tắt nhanh các tính năng:

-   Kiểm soát truy cập web
-   Chặn game/ứng dụng
-   Chụp ảnh màn hình
-   Lập lịch khóa máy tính
-   Lập lịch chặn vào mạng
-   Lập lịch chặn game
