Tính năng này cho phép:

-   Chặn truy cập các trang web khiêu dâm, game online, mạng xã hội,…
-   Có thể tự thêm website cần chặn
-   Cơ sở dữ liệu web được cập nhật tự động từ server KidSafe về
-   Các website được chia thành các nhóm cho dễ quản lý; Một số nhóm đã có sẵn gồm: web khiêu dâm, game online, mạng xã hội,…
-   Có chế độ kiểm soát vừa phải và kiểm soát nghiêm ngặt
